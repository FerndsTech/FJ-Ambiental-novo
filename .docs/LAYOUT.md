# Design System
## Cores
- Primary (Verde Floresta): #064e3b (emerald-900)
- Accent (Verde Neon/Ação): #34d399 (emerald-400)
- Background: #0f172a (slate-900) - Dark Mode Style.
- Text: #f8fafc (slate-50)
## Tipografia
- Headings: Font-Sans (Inter/Roboto), Bold, Tracking-tight.
- Body: Font-Sans, Relaxed reading.
## UX Motion
- GSAP ScrollTrigger: Elementos surgem suavemente de baixo para cima (fade-in-up).
- Micro-interações: Botões aumentam escala (scale-105) no hover.
